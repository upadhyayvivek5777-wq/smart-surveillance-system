\# 🚨 Smart Surveillance System



A real-time surveillance system using OpenCV and Flask.



\## 🔥 Features

\- Motion Detection

\- Image Capture

\- Web Dashboard

\- Responsive UI



\## 🛠️ Tech Stack

\- Python

\- OpenCV

\- Flask

\- HTML, CSS



\## ▶️ How to Run

pip install opencv-python flask  

python main.py  

python app.py  



\## 📸 Output

Detects motion and stores images in alerts folder



\## 🎯 Description

This system detects motion using computer vision and provides a dashboard for monitoring.
